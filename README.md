# Speech to Text

A free, distraction-free speech-to-text web application for long writing sessions.

**Live:** [https://oathanrex.github.io/speech/](https://oathanrex.github.io/speech/)

## Features

- Voice typing with 100+ languages
- Word and character count
- Word target goals
- Export as .txt
- Session history (up to 3 previous sessions)
- Dark mode
- Installable as PWA
- No sign-up required
- No data stored on servers

## Supported Browsers

- Google Chrome (recommended)
- Microsoft Edge
- Brave

## How to Use

1. Open the application
2. Select your language
3. Click "Start Dictation"
4. Allow microphone access
5. Speak clearly
6. Click "Stop Dictation" when finished

### Voice Commands (English only)

| Say this       | Action              |
|----------------|---------------------|
| new line       | Insert a line break |
| stop dictation | Stop the microphone |
| clear text     | Clear all text      |

### Keyboard Shortcuts

- `Ctrl + Shift + D`: Toggle dictation
- `Ctrl + Shift + E`: Export as .txt

## Privacy

- All processing happens in your browser
- Audio is processed by your browser's speech API
- Text is saved only in your browser's local storage
- No data is sent to any server by this application
